<task type="auto">
  <name>Task 1: Create bootstrap.sh first-run script</name>
  <files>bootstrap.sh</files>
  <read_first>
    docker-compose.yml,
    clients/default/client.env,
    clients/default/models.yaml,
    config/litellm/proxy_config.yaml,
    .planning/phases/01-infrastructure-foundation/01-RESEARCH.md
  </read_first>
  <action>
    Create `bootstrap.sh` at project root. Make it executable (chmod +x). Per D-05, D-11, D-12:

    Script structure:

    ```bash
    #!/usr/bin/env bash
    set -euo pipefail
    ```

    **Arguments:**
    - `$1` = client name (default: "default")
    - Derive paths: `ENV_FILE="clients/${CLIENT}/client.env"`, `MODELS_FILE="clients/${CLIENT}/models.yaml"`, `PROXY_CONFIG="config/litellm/proxy_config.yaml"`

    **Step 1 — Validate prerequisites:**
    - Check `docker compose version` succeeds, else error "Docker Compose V2 required"
    - Check ENV_FILE exists, else error with instructions to copy from client.env.example
    - Check MODELS_FILE exists, else error

    **Step 2 — Source client env:**
    - `set -a; source "$ENV_FILE"; set +a`

    **Step 3 — Generate WEBUI_SECRET_KEY if placeholder:**
    - If WEBUI_SECRET_KEY equals "CHANGE_ME_RUN_BOOTSTRAP" or is empty:
      - Generate: `WEBUI_SECRET_KEY=$(openssl rand -hex 32)`
      - Write back to client.env using sed: `sed -i "s/^WEBUI_SECRET_KEY=.*/WEBUI_SECRET_KEY=${WEBUI_SECRET_KEY}/" "$ENV_FILE"`
      - Print: "Generated WEBUI_SECRET_KEY and saved to $ENV_FILE"

    **Step 4 — GPU detection (per D-11):**
    - Function `detect_profile()`:
      - If `OLLAMA_GPU_ENABLED` is explicitly "false" -> return "cpu"
      - Else if `nvidia-smi &>/dev/null` succeeds -> return "gpu"
      - Else -> return "cpu"
    - `PROFILE=$(detect_profile)`
    - Print: "Detected profile: $PROFILE"

    **Step 5 — CPU model downgrade (per D-12):**
    - If PROFILE is "cpu":
      - Override: `REASONING_MODEL="qwen3:7b"` (instead of 14b)
      - Override: `CLASSIFIER_MODEL="gemma3:1b"` (instead of 4b)
      - EMBEDDING_MODEL stays the same (nomic-embed-text)
      - Print: "CPU profile: using smaller models (qwen3:7b, gemma3:1b)"

    **Step 6 — Generate proxy_config.yaml:**
    - `mkdir -p "$(dirname "$PROXY_CONFIG")"`
    - Write proxy_config.yaml using heredoc with variable substitution:
      ```yaml
      model_list:
        - model_name: reasoning-model
          litellm_params:
            model: ollama_chat/${REASONING_MODEL}
            api_base: http://ollama:11434
            api_key: none
        - model_name: classifier-model
          litellm_params:
            model: ollama_chat/${CLASSIFIER_MODEL}
            api_base: http://ollama:11434
            api_key: none
        - model_name: embedding-model
          litellm_params:
            model: ollama/${EMBEDDING_MODEL}
            api_base: http://ollama:11434
            api_key: none
      general_settings:
        master_key: ${LITELLM_MASTER_KEY}
      litellm_settings:
        drop_params: true
      ```

    **Step 7 — LiteLLM version pre-flight check:**
    - Run: `docker pull ghcr.io/berriai/litellm:main-latest`
    - Run: `LITELLM_VERSION=$(docker run --rm ghcr.io/berriai/litellm:main-latest litellm --version 2>/dev/null || echo "unknown")`
    - Print: "LiteLLM version: $LITELLM_VERSION"
    - Print warning if version cannot be determined (do not fail — version check is best-effort)

    **Step 8 — Start Ollama only and wait for ready:**
    - `docker compose --profile "$PROFILE" up -d ollama-gpu ollama-cpu 2>/dev/null || docker compose --profile "$PROFILE" up -d`
    - Wait loop: `until wget -qO- http://localhost:11434/api/tags &>/dev/null 2>&1; do sleep 2; done` with a 120-second timeout counter that errors if exceeded
    - Print: "Ollama is ready"

    **Step 9 — Pull models:**
    - For each model in REASONING_MODEL, CLASSIFIER_MODEL, EMBEDDING_MODEL:
      - Print: "Pulling $MODEL..."
      - `docker exec ollama ollama pull "$MODEL"`
    - Print: "All models pulled"

    **Step 10 — Start full stack:**
    - `docker compose --profile "$PROFILE" up -d`
    - Print: "Stack started successfully"
    - Print: "Open WebUI: http://localhost:${WEBUI_PORT:-3000}"
    - Print: "LiteLLM:    http://localhost:4000"
    - Print: "Ollama:     http://localhost:11434"

    Make the script executable: `chmod +x bootstrap.sh`

    IMPORTANT implementation notes:
    - Use `wget` not `curl` for health polling (matching Ollama image constraints)
    - All inter-container references use service names (ollama, litellm), NOT localhost
    - The localhost references in the wait loop are for HOST-side polling (port-forwarded), which is correct
    - Do NOT use `docker-compose` (standalone binary) — use `docker compose` (V2 plugin)
    - Use `set -euo pipefail` for safe bash execution
