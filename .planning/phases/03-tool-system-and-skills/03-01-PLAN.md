```python
from enum import Enum
from dataclasses import dataclass
from typing import Optional
from pydantic import BaseModel, Field
import numpy as np

class SkillDef(BaseModel):
    """Schema for a single skill YAML file. Per D-02, includes everything inline."""
    name: str
    description: str
    triggers: list[str] = Field(default_factory=list)
    autonomy: str = "confirm-first"  # Per D-12: default is confirm-first
    tools: list[str] = Field(default_factory=list)
    agent: dict
    task: dict

class RoutingDecision(Enum):
    AUTO_RUN = "auto_run"
    CONFIRM_FIRST = "confirm_first"
    FREEFORM = "freeform"
    LIST_SKILLS = "list_skills"

class MatchResult:
    def __init__(self, decision: RoutingDecision, skill: Optional[SkillDef] = None, score: float = 0.0):
        self.decision = decision
        self.skill = skill
        self.score = score

@dataclass
class SkillIndex:
    skills: list[SkillDef]
    embeddings: np.ndarray  # shape (N, D), L2-normalized
```

**Step 1: Create `src/core_api/skills/registry.py`** — implements skill YAML discovery and embedding index per D-01, D-02, D-03, D-05:

- Import `os, httpx, numpy as np, yaml` and `from pathlib import Path`, `from typing import Optional`.
- Import `from skills.models import SkillDef, SkillIndex`.
- Import `from logging_config import get_logger; logger = get_logger(__name__)`.
- Constants: `OLLAMA_BASE = os.getenv("OLLAMA_BASE_URL", "http://ollama:11434")` and `EMBEDDING_MODEL = os.getenv("EMBEDDING_MODEL", "nomic-embed-text")`.
- Module-level `_INDEX: Optional[SkillIndex] = None`.

- `_embed_texts(texts: list[str]) -> np.ndarray`: Calls Ollama `/api/embed` endpoint via `httpx.post(f"{OLLAMA_BASE}/api/embed", json={"model": EMBEDDING_MODEL, "input": texts}, timeout=30.0)`. Returns L2-normalized numpy array of shape `(len(texts), D)`. Response JSON has `{"embeddings": [[...], [...]]}`. Normalize each vector: `vec / np.linalg.norm(vec)`. IMPORTANT: Use `/api/embed` NOT `/api/embeddings` per Research Pitfall 6.

- `_warmup_embedding_model() -> None`: Call `_embed_texts(["warmup"])` to force Ollama to load the embedding model before processing skills. Log at INFO: `"Embedding model warmup complete"`. Per Research Pitfall 2, this prevents slow startup.

- `load_skills(skills_dir: Path, allowed_tools: Optional[set[str]] = None) -> SkillIndex`: Iterate `sorted(skills_dir.glob("*.yaml"))`, parse each with `yaml.safe_load()`, validate with `SkillDef(**raw)`. If `allowed_tools is not None`, filter `skill.tools` to only those in `allowed_tools`. Build embedding text per skill as `skill.name + " " + skill.description + " " + " ".join(skill.triggers)`. Collect all embed texts into a list, call `_embed_texts(all_texts)` once (batch). If no skills found, return `SkillIndex(skills=[], embeddings=np.empty((0, 768), dtype=np.float32))`. Log each indexed skill at INFO.

- `get_index() -> Optional[SkillIndex]`: Returns `_INDEX`.

- `initialize(skills_dir: Path, allowed_tools: Optional[set[str]] = None) -> None`: Calls `_warmup_embedding_model()`, then sets `_INDEX = load_skills(skills_dir, allowed_tools)`. Log count: `"Skill registry initialized: %d skills"`.

**Step 2: Create `src/core_api/skills/matcher.py`** — implements three-zone routing per D-05, D-06, D-07, D-14:

- Import `os, numpy as np`.
- Import `from skills.models import RoutingDecision, MatchResult`.
- Import `from skills.registry import get_index, _embed_texts`.
- Import `from logging_config import get_logger; logger = get_logger(__name__)`.

- Constants:
  ```python
  HIGH_THRESHOLD = float(os.getenv("SKILL_HIGH_THRESHOLD", "0.7"))
  LOW_THRESHOLD = float(os.getenv("SKILL_LOW_THRESHOLD", "0.5"))
  LIST_SKILLS_KEYWORDS = frozenset(["list skills", "what can you do", "show skills", "available skills"])
  ```

- `route(user_message: str) -> MatchResult`:
  1. `lower = user_message.lower().strip()`
  2. Check if `any(kw in lower for kw in LIST_SKILLS_KEYWORDS)` — if yes, return `MatchResult(RoutingDecision.LIST_SKILLS)` per D-14.
  3. Get index via `get_index()`. If None or empty skills, return `MatchResult(RoutingDecision.FREEFORM)`.
  4. Embed user message: `query_vec = _embed_texts([user_message])[0]` (single text, take first vector).
  5. Compute scores: `scores = index.embeddings @ query_vec` (dot product of L2-normalized vectors = cosine similarity).
  6. `best_idx = int(np.argmax(scores))`, `best_score = float(scores[best_idx])`, `skill = index.skills[best_idx]`.
  7. Log: `"Skill match: skill=%s score=%.3f"`.
  8. Three-zone logic per D-06:
     - `best_score >= HIGH_THRESHOLD`: If `skill.autonomy == "auto-execute"` return `MatchResult(RoutingDecision.AUTO_RUN, skill, best_score)`, else return `MatchResult(RoutingDecision.CONFIRM_FIRST, skill, best_score)`.
     - `best_score >= LOW_THRESHOLD`: Return `MatchResult(RoutingDecision.CONFIRM_FIRST, skill, best_score)`.
     - Otherwise: Return `MatchResult(RoutingDecision.FREEFORM, skill=None, score=best_score)`.
