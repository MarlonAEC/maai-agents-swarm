1. `init_embed_model() -> None`:

   - Create and return `QdrantClient(host=os.getenv("QDRANT_HOST", "qdrant"), port=int(os.getenv("QDRANT_PORT", "6333")))`.

3. `_collection_name(client_id: str) -> str`:
   - Return `f"maai_{client_id}_documents"` — per D-05 per-client isolation.

4. `index_document(client_id: str, pages: list[dict], file_name: str) -> int`:
   - Takes `pages` as list of dicts `{"page_no": int, "text": str}` (from docproc response).
   - Creates `Document` objects from `llama_index.core` for each page, with `metadata={"file_name": file_name, "page_label": str(page["page_no"]), "client_id": client_id}`.
   - Creates `SemanticSplitterNodeParser` with `buffer_size=1, breakpoint_percentile_threshold=95, embed_model=Settings.embed_model` (per D-07).
   - Calls `splitter.get_nodes_from_documents(documents)` to get nodes.
   - Creates `QdrantVectorStore(client=_get_qdrant_client(), collection_name=_collection_name(client_id))`.
   - Creates `StorageContext.from_defaults(vector_store=vector_store)`.
   - Creates `VectorStoreIndex(nodes, storage_context=storage_context)`.
   - Returns `len(nodes)` (number of chunks indexed).
   - Log "Indexed {n} chunks for file={file_name} client={client_id} collection={collection_name}".

5. `query_documents(client_id: str, question: str, top_k: int = 5) -> list[dict]`:
   - Restores index from existing collection: `VectorStoreIndex.from_vector_store(vector_store=QdrantVectorStore(client=_get_qdrant_client(), collection_name=_collection_name(client_id)))`.
   - Creates retriever: `index.as_retriever(similarity_top_k=top_k)`.
   - Calls `retriever.retrieve(question)`.
   - Returns list of dicts: `[{"text": node.text, "score": node.score, "file_name": node.metadata.get("file_name", "unknown"), "page_label": node.metadata.get("page_label", "?")} for node in nodes]`.
   - Log "Retrieved {n} chunks for client={client_id} question length={len(question)}".

All functions use `get_logger(__name__)` for logging.

**src/core_api/rag/gpu_lock.py** — Redis-based GPU semaphore per D-12 and D-13:

```python
import redis.asyncio as aioredis
from logging_config import get_logger

logger = get_logger(__name__)

GPU_LOCK_KEY = "maai:gpu_lock"
GPU_LOCK_TTL_MS = 120_000  # 2 minutes max hold time

async def acquire_gpu_lock(
    redis_client: aioredis.Redis,
    ttl_ms: int = GPU_LOCK_TTL_MS,
    max_retries: int = 30,
    retry_delay: float = 1.0,
) -> bool:
    """Try to acquire the GPU lock with retries.
    
    Per D-12: Only one GPU workload at a time.
    Per D-13: Chat has priority — worker should retry patiently.
    Returns True if acquired, False if all retries exhausted.
    """
    import asyncio
    for attempt in range(max_retries):
        result = await redis_client.set(GPU_LOCK_KEY, "1", nx=True, px=ttl_ms)
        if result is True:
            logger.info("GPU lock acquired (attempt %d)", attempt + 1)
            return True
        await asyncio.sleep(retry_delay)
    logger.warning("GPU lock not acquired after %d retries", max_retries)
    return False

async def release_gpu_lock(redis_client: aioredis.Redis) -> None:
    """Release the GPU lock."""
    await redis_client.delete(GPU_LOCK_KEY)
    logger.info("GPU lock released")
```

1. `async def startup(ctx: dict) -> None`:
   - Create `ctx["http_client"] = httpx.AsyncClient(timeout=300.0)` (5min timeout for large docs)
   - Create `ctx["redis"] = aioredis.from_url(f"redis://{os.getenv('REDIS_HOST', 'redis')}:{os.getenv('REDIS_PORT', '6379')}")`.
   - Log "Ingest worker started".

2. `async def shutdown(ctx: dict) -> None`:
   - `await ctx["http_client"].aclose()`
   - `await ctx["redis"].aclose()`
   - Log "Ingest worker shut down".

3. `async def process_document(ctx: dict, file_path: str, client_id: str, file_name: str) -> dict`:
   - This is the main ARQ job function. First arg is always `ctx`.
   - Log "Processing document: file_name={file_name} client={client_id}".
   - **Step 1: Acquire GPU lock** — call `acquire_gpu_lock(ctx["redis"])`. If False, return `{"status": "error", "detail": "GPU lock timeout"}`.
   - **Step 2: Call docproc** — POST `http://{os.getenv('DOCPROC_URL', 'http://docproc:8001')}/process` with JSON `{"file_path": file_path, "ocr_enabled": True}`. Use `ctx["http_client"]`.
   - **Step 3: Release GPU lock** — call `release_gpu_lock(ctx["redis"])` in a finally block (GPU lock covers the docproc call since that's where OCR runs on GPU).
   - **Step 4: Parse response** — if status != "success", return `{"status": "error", "detail": resp_data.get("detail", "docproc failed")}`.
   - **Step 5: Index document** — call `index_document(client_id=client_id, pages=resp_data["pages"], file_name=file_name)`. This is CPU-bound (embedding calls Ollama, not local GPU), so no GPU lock needed.
   - **Step 6: Return** — `{"status": "complete", "chunks": chunk_count, "file_name": file_name}`.
   - Wrap steps 2-5 in try/except with GPU lock release in finally block.

4. `class WorkerSettings`:
   - `functions = [process_document]`
   - `redis_settings = RedisSettings(host=os.getenv("REDIS_HOST", "redis"), port=int(os.getenv("REDIS_PORT", "6379")))`
   - `on_startup = startup`
   - `on_shutdown = shutdown`
   - `max_jobs = 1` — per D-12, one document at a time for GPU sequencing
   - `job_timeout = 600` — 10 minutes max per document (large scanned PDFs are slow)
   - `keep_result = 3600` — keep results 1 hour for status polling (D-03)

All functions use `get_logger(__name__)`.
